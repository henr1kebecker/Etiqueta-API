## Get started

1. Active your virtual environment

    For example:
   ```bash
   ./env\Scripts\activate
   ```

2. Install dependencies though requirements file.

   ```bash
    py install -m requirements.txt
   ```

3. Start project:

    ```bash
    py manage.py runserver
    ```
    or 
    ```bash
    py manage.py runserver 0.0.0.0:8000
    ```
